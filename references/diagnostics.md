# Diagnostics — 4-Tier Maximal Diagnostics Menu (China-Specific)

## Tier 1: 必做 (Essential) — ¥3,000–10,000

### Gene Panel Testing (¥3,000–8,000, 7–14 days)
- **What it tells you:** Actionable mutations (EGFR, ALK, ROS1, BRAF, KRAS, HER2, etc.) — the targets that have approved drugs.
- **How it changes decisions:** Directly determines targeted therapy eligibility. No panel = blind treatment.
- **Where in China:** 燃石医学 (Burning Rock), 华大基因 (BGI), 泛生子 (Genetron), 世和基因 (Geneseeq), 至本医疗 (OrigiMed)
- **Note:** Panel size matters — 300+ genes preferred over small panels. Request TMB and MSI from the same panel.

### Standard IHC (¥500–2,000, 3–7 days)
- **What it tells you:** Protein expression levels — ER/PR/HER2 (breast), PD-L1, MMR proteins, Ki-67 proliferation index.
- **How it changes decisions:** HER2 status determines trastuzumab; MMR/MSI determines immunotherapy; Ki-67 guides chemo aggressiveness.
- **Where:** Any tertiary hospital pathology department (三甲医院病理科).

### Basic Tumor Markers (¥200–500, 1–3 days)
- **What it tells you:** Baseline levels for monitoring (CEA, AFP, CA-125, CA19-9, PSA, etc. depending on cancer type).
- **How it changes decisions:** Establishes baseline for treatment response tracking. Elevated markers + imaging = stronger progression signal.
- **Where:** Any hospital lab.

## Tier 2: 强烈建议 (Strongly Recommended) — ¥10,000–30,000

### Whole Exome Sequencing / WES (¥8,000–15,000, 14–21 days)
- **What it tells you:** All coding mutations (~22,000 genes), accurate TMB, neoantigen landscape, HRD score.
- **How it changes decisions:** Finds rare/novel targets panels miss. TMB-high → immunotherapy. Neoantigen data → vaccine candidacy. HRD → PARP inhibitor.
- **Where:** 燃石, 华大, 世和, 诺禾致源 (Novogene), 药明奥测 (WuXi)

### RNA-seq (¥8,000–15,000, 14–21 days)
- **What it tells you:** Gene fusions (especially novel ones panels miss), gene expression signatures, immune microenvironment profiling.
- **How it changes decisions:** Catches fusions that DNA panels miss (especially in sarcoma, NSCLC). Expression signatures can predict chemo sensitivity.
- **Where:** 燃石, 泛生子, 诺禾致源. Requires RNA-quality tissue (fresh-frozen preferred).

### ctDNA / MRD Baseline (¥5,000–15,000, 7–14 days)
- **What it tells you:** Circulating tumor DNA level — a liquid biopsy snapshot of tumor burden and clonal evolution.
- **How it changes decisions:** Establishes baseline for MRD monitoring post-treatment. Detects resistance mutations emerging before imaging shows progression. Can guide treatment switch 2–3 months earlier than CT.
- **Where:** 燃石 (brPROGRESS), 世和, 华大, 臻和科技 (Geneplus)

### PD-L1 Multi-Platform (¥2,000–5,000, 5–7 days)
- **What it tells you:** PD-L1 expression level — but results vary by antibody clone (22C3, 28-8, SP142, SP263) and scoring system (TPS, CPS, IC).
- **How it changes decisions:** High PD-L1 → immunotherapy first-line in many cancers. Different drugs reference different clones — test the relevant one.
- **Where:** Hospital pathology + send-out to certified labs.

## Tier 3: 深度探索 (Deep Exploration) — ¥30,000–100,000

### Whole Genome Sequencing / WGS (¥15,000–30,000, 21–30 days)
- **What it tells you:** Full genome including non-coding regions, structural variants, SVs, copy number landscape, mutational signatures (COSMIC).
- **How it changes decisions:** Identifies structural rearrangements missed by WES. Mutational signatures reveal DNA repair defects → synthetic lethality targets.
- **Where:** 诺禾致源, 华大, 安诺优达 (Annoroad). Requires high-quality DNA + sufficient tumor content.

### Single-Cell Sequencing (¥30,000–80,000, 21–45 days)
- **What it tells you:** Tumor heterogeneity at cellular resolution — subclone composition, immune cell states, T-cell exhaustion profiles, tumor microenvironment architecture.
- **How it changes decisions:** Identifies resistant subclones before they dominate. Reveals whether immune cells are functional or exhausted → guides immunotherapy combinations.
- **Where:** 新格元 (Singleron), 百奥智汇 (BioMarker), 华大 (DNBelab C4). Requires fresh tissue (process within hours of collection).

### Organoid Drug Sensitivity Testing (¥30,000–60,000, 4–8 weeks)
- **What it tells you:** Which drugs actually kill YOUR tumor cells ex vivo — functional testing beyond genomic prediction.
- **How it changes decisions:** Validates genomic predictions. Tests combinations. Particularly valuable when multiple options exist or genomics is ambiguous.
- **Where:** 丹望医疗 (Danwang), 科途医学 (KeTu), 创芯国际 (Cregene), some top hospitals (中山肿瘤, 复旦肿瘤). Requires fresh tumor tissue.

### TCR Sequencing (¥10,000–20,000, 14–21 days)
- **What it tells you:** T-cell receptor repertoire diversity and clonality — are your immune cells recognizing the tumor?
- **How it changes decisions:** High clonality pre-treatment → better immunotherapy response prediction. Tracks immune response to treatment. Baseline for immunotherapy monitoring.
- **Where:** 艾吉泰康 (ImmuQuad), 华大, 燃石.

## Tier 4: 前沿 (Frontier) — ¥100,000+

### Multi-Timepoint Single-Cell (¥100,000–300,000, months)
- **What it tells you:** How tumor and immune landscape evolve over treatment — clone dynamics, immune adaptation, resistance mechanisms in real-time.
- **How it changes decisions:** Sid did this at 3 timepoints — revealed T-cell infiltration going from 19% to 89%. Guides treatment sequencing and combination timing.
- **Where:** Same as single-cell above, but requires serial fresh biopsies (pre-treatment, mid-treatment, post-treatment).

### Comprehensive MRD Monitoring (¥50,000–100,000/year, serial draws)
- **What it tells you:** Ultra-sensitive detection of residual disease. Can detect 1 tumor cell per million — months before imaging progression.
- **How it changes decisions:** MRD-negative → may safely de-escalate. MRD-rising → pre-emptive treatment switch. Transforms follow-up from reactive to proactive.
- **Where:** 燃石 (brPROGRESS), 世和, 臻和. Requires baseline tumor profile first.

### Full Immune Profiling (¥50,000–150,000, 21–45 days)
- **What it tells you:** Complete immune system characterization — CyTOF/mass cytometry, spatial transcriptomics, multiplex IF, cytokine profiling. Where exactly are immune cells in relation to tumor? What state are they in?
- **How it changes decisions:** Spatial context reveals immune exclusion vs. infiltration patterns → guides immunotherapy strategy. Cold tumor → consider converting strategies first.
- **Where:** Specialized labs — 裕策生物 (YuceBio), research hospital labs (北京大学肿瘤医院, 中山大学肿瘤防治中心).

## Critical Sample Preservation Protocol

> **This was Sid Sijbrandij's key learning — get this right at the FIRST biopsy/surgery. You cannot go back.**

### Rules:
1. **Request fresh-frozen tissue, NOT just FFPE**
   - FFPE (formalin-fixed paraffin-embedded) degrades RNA and limits single-cell analysis
   - Fresh-frozen preserves RNA, proteins, and cell viability
   - Tell surgeon/pathologist BEFORE procedure: "我需要新鲜冷冻组织标本，不仅仅是石蜡包埋"

2. **Split into >=3 aliquots:**
   - Aliquot 1: FFPE (standard pathology — non-negotiable)
   - Aliquot 2: Fresh-frozen (-80C or liquid nitrogen) for genomics/transcriptomics
   - Aliquot 3: Fresh in culture medium (on ice) for organoid establishment and/or single-cell
   - Bonus: Additional fresh-frozen for future unknown tests

3. **Paired blood sample:**
   - 10ml EDTA tube → germline DNA (distinguishes somatic vs. germline mutations)
   - 10ml Streck/cfDNA tube → baseline ctDNA
   - 5ml heparin tube → PBMC isolation (immune profiling, TCR baseline)

4. **Document preservation conditions:**
   - Time from excision to preservation (ischemia time — shorter is better, <30 min ideal)
   - Storage temperature and method
   - Tumor content estimate (pathologist assessment)
